## 이 책에 대하여

이 책은 보이지 않는 세계를 이성적으로, 그리고 영적으로 동시에 탐험합니다.

### 핵심 주장

1. 인간은 4차원(3D 공간 + 시간) 세계에 살고 있습니다.
2. 하나님은 11차원 또는 그 이상의 고차원에 존재하십니다.
3. 3차원에 갇힌 인간은 고차원의 하나님을 볼 수 없습니다.
4. 예수님은 고차원에서 저차원으로 오셔서 우리와 하나님을 연결하셨습니다.
5. 믿음은 차원을 연결하는 다리입니다.

### 대상 독자

- 예수님을 아직 모르지만 과학적 논리와 성경적 진리를 함께 접하고 싶은 분
- 이미 믿음을 가졌지만, 과학적 해석과 성경적 해석을 병행하여 신앙을 강화하고 싶은 분
