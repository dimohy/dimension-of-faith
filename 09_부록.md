## 부록 A: 차원 이해를 위한 추가 설명

### 차원의 수학적 정의

차원(Dimension)은 수학적으로 **독립적인 좌표의 수**를 의미합니다. 어떤 공간에서 한 점의 위치를 지정하기 위해 필요한 숫자의 개수가 그 공간의 차원입니다.

| 차원 | 필요한 좌표 수 | 예시 |
|------|--------------|------|
| 0차원 | 0개 | 점 |
| 1차원 | 1개 (x) | 선, 직선 위의 점 |
| 2차원 | 2개 (x, y) | 면, 종이 위의 점 |
| 3차원 | 3개 (x, y, z) | 공간, 방 안의 점 |
| 4차원 | 4개 (x, y, z, t) | 시공간, 특정 시간과 장소 |

### 플랫랜드(Flatland) 상세 설명

**플랫랜드: 다차원 로맨스(Flatland: A Romance of Many Dimensions)**는 1884년 영국의 교육자 에드윈 애벗(Edwin Abbott Abbott)이 쓴 소설입니다.

**줄거리 요약:**
- 주인공 "정사각형"은 2차원 세계 플랫랜드에 삽니다.
- 어느 날 3차원 존재인 "구"가 방문합니다.
- 정사각형은 처음에 구를 이해하지 못하지만, 3차원으로 들어 올려져 새로운 세계를 경험합니다.
- 플랫랜드로 돌아온 정사각형은 3차원의 존재를 전파하려 하지만, 아무도 믿지 않아 감옥에 갇힙니다.

**교훈:**
- 저차원 존재는 고차원을 상상하기 어렵습니다.
- 고차원 존재가 직접 와서 보여주지 않으면 이해할 수 없습니다.
- 새로운 차원의 진리를 말하면 기존 세계에서 이상하게 취급받을 수 있습니다.

### 끈 이론과 M이론 간략 소개

**끈 이론(String Theory)**
- 우주의 기본 단위가 점 입자가 아니라 진동하는 "끈"이라는 이론
- 끈의 진동 패턴에 따라 다른 입자로 나타남
- 수학이 작동하려면 10차원이 필요

**M이론(M-Theory)**
- 1995년 에드워드 위튼이 제안
- 여러 끈 이론을 통합하는 더 큰 이론
- 11차원을 요구함
- "M"의 의미는 명확히 정해지지 않음 (Mother, Master, Mystery, Membrane 등 여러 해석)

### 시간의 특수성

시간은 공간과 다른 특징을 가집니다:

| 공간 | 시간 |
|------|------|
| 앞뒤로 자유롭게 이동 | 한 방향으로만 흐름 |
| 원하는 곳에 머물 수 있음 | 멈출 수 없음 |
| 3차원 | 1차원 |

물리학에서는 이 차이를 **"시간은 허수 좌표를 가진다"**고 표현하기도 합니다. 시공간을 다룰 때 시간 좌표에 허수 i를 곱하면 수학이 더 깔끔해집니다.

## 부록 B: 참고 성경 구절 모음

### 차원과 관련된 구절

**보이지 않는 세계**
> *"우리가 주목하는 것은 보이는 것이 아니요 보이지 않는 것이니 보이는 것은 잠깐이요 보이지 않는 것은 영원함이라"* — 고린도후서 4:18

> *"믿음은 바라는 것들의 실상이요 보이지 않는 것들의 증거니"* — 히브리서 11:1

**하나님의 초월성**
> *"하나님이 참으로 땅에 거하시리이까 하늘과 하늘들의 하늘이라도 주를 용납하지 못하겠거든"* — 열왕기상 8:27

> *"여호와께서 이르시되 내 생각은 너희의 생각과 다르며 내 길은 너희의 길과 다르니라"* — 이사야 55:8

**하나님을 볼 수 없음**
> *"본래 하나님을 본 사람이 없으되 아버지 품 속에 있는 독생하신 하나님이 나타내셨느니라"* — 요한복음 1:18

> *"오직 그에게만 죽지 아니함이 있고 가까이 가지 못할 빛에 거하시고 어떤 사람도 보지 못하였고 또 볼 수 없는 이시니"* — 디모데전서 6:16

### 예수님의 성육신, 십자가, 부활

**성육신**
> *"말씀이 육신이 되어 우리 가운데 거하시매 우리가 그의 영광을 보니 아버지의 독생자의 영광이요 은혜와 진리가 충만하더라"* — 요한복음 1:14

> *"그는 근본 하나님의 본체시나 하나님과 동등됨을 취할 것으로 여기지 아니하시고 오히려 자기를 비워 종의 형체를 가지사 사람들과 같이 되셨고"* — 빌립보서 2:6-7

**십자가**
> *"죄의 삯은 사망이요 하나님의 은사는 그리스도 예수 우리 주 안에 있는 영생이니라"* — 로마서 6:23

> *"우리가 아직 죄인 되었을 때에 그리스도께서 우리를 위하여 죽으심으로 하나님께서 우리에 대한 자기의 사랑을 확증하셨느니라"* — 로마서 5:8

**부활**
> *"그리스도께서 죽은 자 가운데서 다시 살아나사 잠자는 자들의 첫 열매가 되셨도다"* — 고린도전서 15:20

> *"나는 부활이요 생명이니 나를 믿는 자는 죽어도 살겠고 무릇 살아서 나를 믿는 자는 영원히 죽지 아니하리니"* — 요한복음 11:25-26

### 믿음과 기도

**믿음**
> *"믿음이 없이는 하나님을 기쁘시게 하지 못하나니 하나님께 나아가는 자는 반드시 그가 계신 것과 또한 그가 자기를 찾는 자들에게 상 주시는 이심을 믿어야 할지니라"* — 히브리서 11:6

**기도**
> *"구하라 그리하면 너희에게 주실 것이요 찾으라 그리하면 찾아낼 것이요 문을 두드리라 그리하면 너희에게 열릴 것이니"* — 마태복음 7:7

> *"쉬지 말고 기도하라"* — 데살로니가전서 5:17

### 성령

> *"너희는 너희가 하나님의 성전인 것과 하나님의 성령이 너희 안에 계시는 것을 알지 못하느냐"* — 고린도전서 3:16

> *"오직 성령의 열매는 사랑과 희락과 화평과 오래 참음과 자비와 양선과 충성과 온유와 절제니"* — 갈라디아서 5:22-23

### 천국과 새 창조

> *"또 내가 새 하늘과 새 땅을 보니 처음 하늘과 처음 땅이 없어졌고"* — 요한계시록 21:1

> *"보라 내가 만물을 새롭게 하노라"* — 요한계시록 21:5

> *"사망이 없고 애통하는 것이나 곡하는 것이나 아픈 것이 다시 있지 아니하리니 처음 것들이 다 지나갔음이러라"* — 요한계시록 21:4

## 부록 D: 용어 해설

### 과학 용어

| 용어 | 설명 |
|------|------|
| **차원(Dimension)** | 공간에서 위치를 지정하기 위해 필요한 독립적인 좌표의 수 |
| **끈 이론(String Theory)** | 우주의 기본 단위가 진동하는 끈이라는 이론 |
| **M이론(M-Theory)** | 여러 끈 이론을 통합하는 11차원 이론 |
| **시공간(Spacetime)** | 3차원 공간과 1차원 시간이 결합된 4차원 구조 |
| **엔트로피(Entropy)** | 무질서도. 닫힌 시스템에서 항상 증가함 |
| **양자역학(Quantum Mechanics)** | 원자 이하의 미시 세계를 다루는 물리학 |
| **상대성 이론(Relativity)** | 아인슈타인이 발표한 시공간과 중력에 관한 이론 |
| **양자 얽힘(Quantum Entanglement)** | 두 입자가 거리에 상관없이 연결되어 있는 현상 |
| **웜홀(Wormhole)** | 시공간의 두 지점을 연결하는 가상의 터널 |
| **다중 우주(Multiverse)** | 우리 우주 외에 다른 우주들이 존재한다는 가설 |

### 신학 용어

| 용어 | 설명 |
|------|------|
| **성육신(Incarnation)** | 하나님이 인간의 몸을 입으신 것 (예수 그리스도) |
| **케노시스(Kenosis)** | 자기 비움. 예수님이 신적 특권을 내려놓으신 것 |
| **구속(Redemption)** | 값을 치르고 구해내는 것. 예수님이 우리의 죄 값을 치르심 |
| **칭의(Justification)** | 죄인이 의롭다 인정받는 것 |
| **성화(Sanctification)** | 거룩하게 되어가는 과정 |
| **전지(Omniscience)** | 하나님이 모든 것을 아시는 속성 |
| **전능(Omnipotence)** | 하나님이 모든 것을 하실 수 있는 속성 |
| **편재(Omnipresence)** | 하나님이 모든 곳에 계시는 속성 |
| **영원(Eternity)** | 시간의 제한을 받지 않는 상태 |
| **천국/하나님 나라(Kingdom of God)** | 하나님이 통치하시는 영역 |
| **재림(Second Coming)** | 예수님이 다시 오시는 것 |
| **부활(Resurrection)** | 죽은 자가 다시 살아나는 것 |
| **새 창조(New Creation)** | 하나님이 만물을 새롭게 하시는 것 |

## 부록 E: 토론 질문

이 책을 함께 읽고 토론할 때 사용할 수 있는 질문들입니다.

### Part 1: 차원의 세계

1. 우리가 4차원에 "갇혀" 있다는 것은 어떤 의미일까요?
2. 2차원 존재가 3차원을 이해할 수 없듯이, 우리도 이해할 수 없는 것들이 있을까요?
3. 과학이 11차원을 말한다면, 그것은 신앙과 어떤 관련이 있을까요?

### Part 2: 하나님, 고차원의 존재

4. 완전하신 하나님이 차원에 제한받지 않으신다면, 우리는 어떻게 그분을 알 수 있을까요?
5. "보이지 않는 것이 없는 것이 아니다"라는 말에 동의하시나요? 그 이유는?
6. 하나님이 우리에게 자신을 알리시는 다양한 방법들에는 무엇이 있을까요?

### Part 3: 예수 그리스도

7. 무한하신 하나님이 유한한 인간이 되셨다는 것을 어떻게 이해할 수 있을까요?
8. 십자가가 "차원의 벽을 허물었다"는 것은 어떤 의미일까요?
9. 예수님의 부활의 몸은 우리의 현재 몸과 어떻게 다를까요?

### Part 4: 믿음

10. 믿음이 "보이지 않는 것들의 증거"라면, 그것은 맹목적인 믿음과 어떻게 다를까요?
11. 기도가 정말 하나님께 닿는다고 생각하시나요? 그 이유는?
12. 성령님이 우리 안에 거하신다는 것을 어떻게 경험할 수 있을까요?

### Part 5: 새 하늘과 새 땅

13. 천국이 "어딘가에 있는 곳"이 아니라면, 그것은 무엇일까요?
14. 부활의 몸을 받는다는 것은 어떤 의미일까요?
15. "하나님이 만유의 주로서 만유 안에 계신다"는 것은 어떤 상태일까요?

### 종합 질문

16. 이 책을 읽기 전과 후, 믿음에 대한 생각이 어떻게 달라졌나요?
17. 과학과 신앙이 서로 모순된다고 생각하시나요, 아니면 조화될 수 있다고 생각하시나요?
18. 이 책에서 가장 인상 깊었던 부분은 무엇인가요? 그 이유는?
